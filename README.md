# Project Buttercup Academy

A personal, vintage-fairy learning dashboard for GitHub Pages.

## What it includes

- German and Portuguese learning paths
- iPhone 13 photography path
- CapCut and DaVinci Resolve cinematography path
- Gouache illustration path
- Writing path
- XP and level titles every 3 levels
- Achievements
- Editable notes and journals
- Portfolio planning area
- Google Calendar task links
- Free course and certificate resource links

## How to publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload these files:
   - `index.html`
   - `README.md`
   - `LICENSE`
   - `.gitignore`
3. Go to repository **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Save.
7. Your site will appear at your GitHub Pages URL.

## Saving progress

This version saves your progress in your browser using `localStorage`.

That means:
- It works on GitHub Pages.
- It does not need a backend.
- Progress is saved on the same browser/device.
- Use the backup button to download your progress as JSON.
